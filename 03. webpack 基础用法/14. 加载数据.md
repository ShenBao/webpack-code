# 加载数据

JSON 、CSV、TSV 和 XML 文件

JSON 已内置支持

要导入 CSV、TSV 和 XML，你可以使用 csv-loader 和 xml-loader

npm install --save-dev csv-loader xml-loader

```js
{
    test: /\.(csv|tsv)$/,
    use: [
    'csv-loader'
    ]
},
{
    test: /\.xml$/,
    use: [
    'xml-loader'
    ]
}
```

src/data.xml

```xml
<?xml version="1.0" encoding="UTF-8"?>
<note>
  <to>Mary</to>
  <from>John</from>
  <heading>Reminder</heading>
  <body>Call Cindy on Tuesday</body>
</note>
```
